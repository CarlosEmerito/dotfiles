# Wallpapers

Guarda tus wallpapers en esta carpeta.

## Descargar wallpapers

```bash
# Unsplash
curl -L -o wallpaper.jpg "https://source.unsplash.com/random/1920x1080"

# Wallhaven
curl -L -o wallpaper.jpg "https://wallhaven.cc/random"
```

## Usar con el script

```bash
./scripts/random-wallpaper.sh ~/dotfiles/wallpapers
```

## Fuentes recomendadas
