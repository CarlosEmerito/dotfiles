# Hyprland Dotfiles

My Hyprland configuration for Arch Linux.

## Installation

```bash
git clone https://github.com/CarlosEmerito/dotfiles.git
cd dotfiles
./install.sh
```

## Update

```bash
./update.sh
```

## Copy only configs

```bash
./copy.sh
```

## Keybinds

| Key | Action |
|-----|--------|
| Super + Space | App launcher |
| Super + Return | Terminal |
| Super + Q | Close window |
| Super + E | File manager |
| Super + F | Browser |
| Super + 1-0 | Workspaces |
